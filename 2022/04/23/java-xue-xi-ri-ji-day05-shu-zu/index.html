(1) 数组

  1. 数组介绍

数组可以存放多个同一类型的数据。数组也是一种数据类型，是引用类型。即：数(数据)组(一组)就是一组数据

  1. 数组的使用细节和注意事项

  • 数组是多个相同类型数据的组合，实现对这些数据的统一管理

  • 数组中的元素可以是任何数据类型，包括基本类型和引用类型，但是不能混用。

  • 数组创建后，如果没有赋值，有默认值

    int 0，short 0, byte 0, long 0, float 0.0,double 0.0，char \u0000，boolean false，String null

  • 使用数组的步骤 1. 声明数组并开辟空间 2 给数组各个元素赋值 3 使用数组

  • 数组的下标是从 0 开始的。

  • 数组下标必须在指定范围内使用，否则报：下标越界异常 int [] arr=new int[5]; 则有效下标为 0-4

  • 数组属引用类型，数组型数据是对象(object)

  1. 数组的赋值机制

  • 基本数据类型赋值，这个值就是具体的数据，而且相互不影响。

    int n1 = 2; int n2 = n1;

  • 在引用数据类型下是引用传递，赋的值是地址。

个人理解: 其实java在内存中严格意义上来讲都是值传递,只不过在引用数据类型中是将arr1数组指向的内存空间的地址值拷贝一份然后赋值给arr2, 但是指向同一内存空间, 当改变其中一个变量的值的时候, 对应内存空间里面存储的值就会发生变化, 导致arr1和arr2的值都会发生变化

  1. 数组拷贝

数组拷贝其实就是要在内存中重新开辟一块空间, 然后将原先空间内的值拷贝到新的空间里去

  1. 数组反转

  1. 数组扩容

在数组扩容中暂时使用的方法是重新开辟一片的新的空间然后对值进行添加和拷贝,但是相对来说效率很低,更高效的方法等到链表继续学习

(2) 排序

内部排序:

  • 指将需要处理的所有数据都加载到内部存储器中进行排序。包括(交换式排序法、选择式排序法和插入式排序法)；

外部排序法：

  • 数据量过大，无法全部加载到内存中，需要借助外部存储进行排序。包括(合并排序法和直接合并排序法)。

冒泡排序法:

在解决冒泡排序问题的时候,要注意化繁为简,先死后活的去分析, 将每次比较的结果写出来找到规律, 写出每次循环然后找到共性写出外层循环负责比较多少轮。

二维数组使用细节和注意事项

  • 一维数组的声明方式有: int[] x 或者 int x[]

  • 二维数组的声明方式有: int[][] y 或者 int[] y[] 或者 int y[][]

  • 二维数组实际上是由多个一维数组组成的，它的各个一维数组的长度可以相同，也可以不相同。比如： map[][] 是一个二维数组

    int map [][] = { { 1,2},{ 3,4,5 } }

  • 由 map[0] 是一个含有两个元素的一维数组 ，map[1] 是一个含有三个元素的一维数组构成，我们也称为列数不等的二维数组
